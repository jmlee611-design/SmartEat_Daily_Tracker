원격 공유 설정

여러 기기에서 같은 데이터를 공유하려면 무료 Supabase 계정이 필요해요. 가족 모두가 같은 "가족 코드"를 입력하면 같은 데이터를 함께 봐요.
1 Supabase 계정 만들기
supabase.com에서 가입 후 새 프로젝트를 만드세요. (무료, 신용카드 불필요)
2 데이터 테이블 만들기
Supabase 대시보드 → SQL Editor → 아래 SQL 붙여넣고 실행:
create table family_data ( family_id text primary key, data jsonb not null, updated_at timestamptz default now() ); create table public_profiles ( device_id text primary key, nickname text not null, data jsonb not null, updated_at timestamptz default now() ); create table follows ( follower_id text, followee_id text, created_at timestamptz default now(), primary key (follower_id, followee_id) ); alter table family_data enable row level security; alter table public_profiles enable row level security; alter table follows enable row level security; create policy "Allow all" on family_data for all using (true) with check (true); create policy "Allow all" on public_profiles for all using (true) with check (true); create policy "Allow all" on follows for all using (true) with check (true);
3 API 정보 입력
Supabase 대시보드 → Settings → API에서 복사:
프로젝트 URL
Anon (Public) Key
4 가족 코드
새로 만들면 다른 가족 구성원에게 공유하세요. 가족이 이미 사용 중이면 그 코드를 입력하세요.
